# TicketMaster (TypeScript + Express)

Basic Node.js + TypeScript + Express starter for the TicketMaster project.

Quick start

Install dependencies:

```bash
npm install
```

Run in development (auto-restarts):

```bash
npm run dev
```

Build and run:

```bash
npm run build
npm start
```

API endpoints

- GET / -> greeting
- GET /health -> health check
